# 为什么还要做个人博客

这个站点不是为了追热点，也不是为了把自己包装成某种稳定的人设。

它更像一个缓慢更新的个人角落：记录我正在学的东西，正在想的事情，和一些不太适合塞进平台算法里的表达。

我想保留一点对内容的掌控感。

在这里，页面长什么样、文章怎么组织、哪些话值得留下，都由我自己决定。

# 这里会写什么

- 折腾博客、网站和工具时遇到的问题
- 一些技术学习过程里的记录
- 日常生活里的观察、想法和随笔

# 你会看到什么风格

大概率不会很快，也不会很多。

但我希望这里的内容是真实的、可回看的，而不是只为了发布当下那一刻。
